## Retail Sales Performance Analysis

### Problem
The goal of this project was to analyze retail sales data to identify top-performing product categories and regions.

### Dataset
Retail sales transaction data containing product category, quantity, price, region, and date.

### Tools Used
- Python (Pandas, Matplotlib, Seaborn)

### Key Insights
- Electronics generated the highest total revenue.
- Ontario was the best-performing region.
- High-priced products drive revenue despite lower sales volume.

### Business Recommendations
- Focus marketing on high-margin electronics.
- Prioritize Ontario for premium product launches.

### Outcome
This analysis provides data-driven insights to support strategic marketing and regional expansion decisions.
